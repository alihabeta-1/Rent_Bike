# Rent Bike App (UI Only)

Rent Bike is a modern Flutter mobile application UI designed for renting bicycles.  
This project focuses **entirely on the user interface (UI/UX)** and visual experience, without backend or API integration.

The goal of this project is to practice and demonstrate clean UI implementation, layout structuring, and modern mobile design principles using Flutter.
